Decisions

IDChoice · Rationale
D-001Open external pages in a new tab instead of embedding them. Many publishers block iframe embedding.
D-002Pick client-side from already-loaded notes. Avoids a new API or D1 query for an additive UI view.
D-003Support /read as an SPA boot path. Keeps the page bookmarkable without a second app shell.
D-004Limit candidates to http and https URLs. The reader is for web articles, not arbitrary URI handlers.
