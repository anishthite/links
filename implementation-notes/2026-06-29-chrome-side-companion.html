Summary

Goal
Ship an unpacked Chrome Side Panel extension that posts the active tab URL into Links.
Source
chrome.tabs.query() supplies the active tab; chrome.storage.sync stores the app base URL.
Surface
extension/sidepanel.html with POST /api/links.
Schema
None.
Mobile
Untouched.

API contract

Request

POST /api/links
{ "sourceUrl": "https://example.com/page", "text": "optional note", "tags": ["optional"] }

Response

{ "note": NoteWire, "duplicate"?: true }

UI layout

  1. Current tab: show title and URL, reject non-http URLs before fetch.
  2. Fields: app URL, optional note, optional comma tags.
  3. Action: save button posts to /api/links and displays created vs duplicate.
